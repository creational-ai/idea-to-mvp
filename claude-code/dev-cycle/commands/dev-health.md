# /dev-health

Project health check to verify alignment with north star vision.

**Guide**: `~/.claude/skills/dev-cycle/references/health-guide.md`
**Template**: `~/.claude/skills/dev-cycle/assets/templates/PROJECT_STATE.md`

## Usage

```bash
/dev-health
```

## What It Does

1. Reviews project artifacts (north star, design, results docs)
2. Assesses alignment, production-readiness, gaps, scope, tests
3. Updates PROJECT_STATE.md with latest health check
4. Generates health report with wins, concerns, recommendations

## When to Run

- After completing a task
- End of session
- After major decision
- Every 2-3 steps during active development
- When feeling uncertain

Read the guide. Follow it exactly.
